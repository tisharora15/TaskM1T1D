Interrupt-Based Sense–Think–Act System
This project implements a real-time home security system using interrupt-driven architecture on Arduino Uno. It integrates multiple sensors and periodic timer events following the Sense–Think–Act model.

External interrupt for PIR

Pin change interrupts for buttons

Timer interrupt for periodic ultrasonic measurement

Modular design

 How to Run

Open in Arduino IDE

Select Arduino Uno

Upload code

Open Serial Monitor 

 Hardware Required

Arduino Uno

2 push buttons

PIR sensor

HC-SR04 ultrasonic sensor

LED + 220Ω resistor
